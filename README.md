# cub3d

### 💡 About the project

This project aims to create a graphic engine (ray casting) using a minimal graphic library.  

### 🛠️ Usage

Compile the project using the provided Makefile: ```make```

Run binary with a valid map: ```./cub3d maps/map01.cub```
